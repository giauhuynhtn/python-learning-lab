# Python Learning Lab

A personal collection of small Python exercises and practice scripts from my programming fundamentals coursework. This repository helps me build fluency with core Python concepts through hands-on practice.

## About

This repo contains my solutions and notes from exercises I completed while learning Python fundamentals. Each file focuses on a specific concept (strings, lists, dictionaries, functions, etc.).

## Topics Covered

- Strings and string methods
- Lists and list comprehensions
- Dictionaries
- Functions
- Loops and conditionals
- File handling (add as you learn)
